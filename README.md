# Distributed Rate Limiter Simulator

A systems simulation that compares rate-limiting algorithms under bursty distributed traffic.

## Highlights
- Policy modes: Token Bucket, Leaky Bucket, Fixed Window.
- Multi-node traffic distribution with per-node metrics.
- Live visualization of capacity and queue pressure.
- Throughput/drop-rate analytics with traffic spike testing.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS control panel

## Run
Open `index.html` in your browser.
